# Example Resumes

This directory contains example resumes for testing the ATS Resume Scoring Agent.

## Files

- **good_resume.txt**: A well-formatted resume that should score 85-95 points
- **basic_resume.txt**: A basic resume with room for improvement (60-75 points)

## Usage

You can upload these sample resumes through the web interface to see how the scoring works:

1. Navigate to http://localhost:8000
2. Upload one of these example files
3. Review the score and recommendations
4. Compare different resume formats

## Creating Your Own Test Resumes

When creating test resumes, consider including:

- Contact information (email, phone, LinkedIn)
- Professional summary or objective
- Work experience with dates and descriptions
- Education section
- Skills section with relevant keywords
- Quantifiable achievements (numbers, percentages)
- Action verbs in experience descriptions

Save in TXT, PDF, or DOCX format.
